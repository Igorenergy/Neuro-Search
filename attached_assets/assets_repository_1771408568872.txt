Here is the simplified description for the Assets Repository page.

I have stripped away the complex search bars, filter chips, and advanced sorting shown in the screenshot, focusing purely on a clean, functional File Manager (MVP) structure.

Page Specification: Assets Repository (Lite Version)
Route: /assets
Role: The system's "Long-term Memory." A clean, "Google Drive-like" file manager where users can store, organize, and access all artifacts (PDFs, CSVs, Reports) generated by the AI or uploaded manually.

1. Layout & Structure
Container: Full-height, 2-column layout.

Left Sidebar (Navigation): Fixed width (250px).

Main Content (Canvas): Fluid width, white/gray background.

Header (Global): Standard app header.

Sub-Header (Context): Breadcrumbs only (e.g., Dashboard > Assets Repository > Financial Reports).

2. Left Sidebar (Navigation)
Primary Action: A large, prominent button "Upload / Create" at the top.

Style: White with shadow or Accent Color.

Icon: Plus (+).

Menu Items (Minimalist):

My Assets (Active state).

Recent Files.

Trash / Bin.

Storage Widget (Bottom):

Simple progress bar.

Text: "2.4 GB of 15 GB used".

Link: "Upgrade Storage".

3. Main Content Area (The Grid)
Toolbar (Simplified):

Left: Folder Name (Large Title, e.g., "My Drive").

Right: View Toggle Icons (Grid ⊞ / List ☰). No Search Bar, No Filter Chips.

The Content Grid:

Section 1: Folders

Displayed as rectangular cards.

Visual: Folder Icon (Gray/Manila) + Title + Item Count (e.g., "12 items").

Mock Data: "Competitor Analysis", "Q3 Financials", "Raw Datasets", "Generated Reports".

Section 2: Files

Displayed as cards with a preview thumbnail or generic format icon.

Visual: Icon (PDF/CSV/DOC) + Filename + Date.

Mock Data: nvidia_annual_report.pdf, leads_export_v2.csv.

4. Interaction Patterns
Navigation:

Double Click: Enters a folder (updates Breadcrumbs).

Single Click: Selects the item (highlights with blue border).

Context Menu (Right-Click):

Since there are no complex action bars, all actions live in the right-click menu:

Open

Rename

Download

Delete (Move to Trash).

Empty State:

If a folder is empty, display a large, clean illustration.

Text: "This folder is empty. Drop files here to upload."

5. Visual Style
Vibe: Clean, airy, "Mac OS Finder" feel.

Colors:

Background: #F9FAFB (Light Gray).

Cards: White with subtle border #E5E7EB.

Hover: Slight lift (shadow) or background darkening.

Typography: San Francisco / Inter. Clean and readable.
